Проект WEB-site + API.

Проект состоит из онлайн магазина с базой данных, а также из сообщества в vk с ботом,
который имеет команды для помощи людому пользователю.

base.html - форма шапки(основная форма)
login.html - форма авторизации
register.html - форма регистрации
catalog.html - форма для основной страницы
profile.html - форма  для профиля авторизированого пользователя
goods_profile.html - форма для профидя продукта
cart.html - форма для корзины
add_goods.html - форма для добавления продукта(только для админа)

server.py  - сервер
vk.py - вк бот

В папке db размещена база данных, в которой есть три таблицы(goods(продукты), orders(заказы), users(пользователи))
Сайт размещен на heroku. Сайт: online-shop-kirill.herokuapp.com
Сначало пользователь попадает на главную страницу со всеми продуктами, которые поделенны на категории.
Если зарегистрироваться и авторезироваться, то появляется корзина и профиль. Можно добавлять продукты в корзину
и оформлять заказы(заказы появляются в таблице orders).

К сайту подключен css: style.css

Также на сайте есть ссылка на сообщество vk, где есть бот, у которого можно получить информацию о сайте,
а также о пользователе, если указать логин.

Добавление продукта доступно только админу.
